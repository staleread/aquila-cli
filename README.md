# aquila-cli

```
Usage: aquila-cli <command>

Flags:
  -h, --help    Show context-sensitive help.

Commands:
  gen [flags]
    Generate a new key pair.

  enc --input=STRING --output=STRING --key=STRING
    Encrypt a file.

  dec --input=STRING --output=STRING --key=STRING
    Decrypt a file.

  config
    Show cipher configuration.

  build <config-id>
    Build the CLI with the specified cipher configuration.
```
